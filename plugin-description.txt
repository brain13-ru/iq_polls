Plugin IQ Polls is intendent for tests and polls creation, mainly for IQ tests and tests for erudition. It is very simple for using. For installing it just copy plugin files to plugin directory of your site. Then it will be possible to activate plugin in administrative panel of your site. After activating new type of post will appear - Polls and tests. Now you can add new test, add questions and answer choices for the questions. You can display your test in front side of your site as you usually do it for others type of posts. Now users can answer the questions of the test. 
As well plugin contain two widgets - for test users top displaying and for current user archive of tests displaying.
Плагин IQ Polls предназначен для создания тестов и опросов, главным образом тестов на интеллект и эрудицию. Он очень прост в использовании. Чтобы установить егопросто скопируйте файлы плагина в каталог плагинов вашего сайта. Посл этого станет возможным активировать плагин в административной панели вашего сайта. После активации появится новый тип записи - Тесты и опросы. Теперь вы можете создать новый тест, добавить в него вопросы и варианты ответов на них. Вы можете вывести ваш тест в пользовательскую часть сайта так же, как вы делаете это с другими типами записей. Теперь пользователи могут отвечать на вопросы теста.
Так же плагин содержит 2 виджета - для вывода рейтинга пользоватеей тестов и вывода архива тестов текущего пользователя.
